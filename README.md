# Пояснительная записка к архиву публикаций Телеграм-канала "Елена Панина"

Приветствую!

В [архиве](docs) — **12.070 файлов** в формате MD (Markdown), позволяющем оформлять текст в хорошо читаемом машинами виде. Markdown — традиционный язык разметки, его документацию можно изучить, например, [здесь](https://markdownguide.offshoot.io/).

Каждый файл начинается с краткой сводки — frontmatter с интуитивно понятными ключами (в [конфигурации TOML](https://toml.io/en/)). Пример сводки:

```toml
+++
title = "Фицо пообещал блокировать членство Украины в НАТО"
id = 15062
date = 2024-10-07T18:32:04
source = "https://t.me/EvPanina/15062"
length = 1764
+++
```

Репосты и пустые посты отсутствуют. Реклама по возможности убрана. Остальное, даже с минимальным текстом, сохранено.

Все MD-файлы можно скачать в виде [одного ZIP-архива](export/TG_Елена_Панина.zip).
