В новый санкционный список ЕС входят "биде, унитазы, смывные бачки и аналогичная сантехника", а также светодиоды, конопляная пряжа, вилочные погрузчики, машины для сортировки почты, дымоходы, кирпичи, шины и "колпачки для шариковых ручек".

*"Туалеты стали символом того, что 24-летнее правление президента РФ Владимира Путина ничего не сделало для улучшения жизни россиян"*, — с бессмысленной настойчивостью [разжигает](https://euobserver.com/world/156725) EUobserver.

Унитазное эмбарго нанесёт мощный удар по России, уверено издание. Ведь такой авторитетный источник, как сам Зеленский, уверял, будто русские мало знакомы с сантехникой и биде для них — вещь неизмеримой ценности...

**Евросоюз опустился уже до сортирного уровня.**
