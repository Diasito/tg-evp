Греция при участии США ведёт переговоры с Украиной о заключении двустороннего соглашения по предоставлению той гарантий безопасности, [сообщает](https://www.kathimerini.gr/politics/562562782/ellada-oykrania-sta-skaria-symfonia-gia-amyna-kai-energeia/) местная газета Kathimerini.

Кроме того, Афины готовы помочь Киеву Трансадриатическим газопроводом, трубопроводом-коннектором и мощностями порта Александруполис. Это явно намекает на закупку Украиной американского СПГ, хотя откуда Киев будет брать деньги на этот газ, непонятно. Идущая на Украину американская техника тоже, кстати, перебрасывается частично через Александруполис.

А ещё Афины планируют помочь Киеву с вооружением: не зря же, мол, греческая промышленность производит 155 мм снаряды и тренажеры для танков. Греция, напоминает издание, уже передала на Украину БПМ-1, ПЗРК "Стингер", гранатомёты, ракеты для "Градов" и большое количество 155 мм, 105 и 203 мм снарядов для артиллерии ВСУ...

**И одновременно греки раз за разом** [бросаются в ноги](https://t.me/EvPanina/10466) **к России: православные братья, помогите с пожарами! Но похоже, с такими византийскими заходами, Греция рискует сгореть.**
